# 🏏 Cricket_Performance_&_Player_Insights_Dashboard
An interactive Power BI dashboard that provides deep insights into cricket player and team performance using visual analytics.

## 📸 Dashboard Preview
👉 
## 📊 Features
### Player Analysis:
• Top 10 run scorers
• Strike rate analysis (scatter chart)
• All-rounder performance

### Team Analysis: 
• Total runs by team
• Balls faced distribution
• Wickets taken by country

### Key Metrics (KPIs):
• Total Runs
• Total Wickets
• Total Players
• Total Matches
• Top Team

### Filters:
• Stage
• Venue
• Country
• City

## 📈 Visualizations Used
• Bar Charts
• Donut & Pie Charts
• Scatter Plot
• KPI Cards
• Line / Area Chart

## 🛠 Tools & Technologies
• Power BI
• DAX
• Data Modeling

## 📌 Key Insights
• Identifies high-performing players using strike rate vs runs
• Compares team performance across multiple metrics
• Provides interactive filtering for deeper analysis

## 🚀 Learnings
• Built dynamic dashboards using Power BI
• Created custom measures using DAX
• Improved data storytelling and UI design
